# 04 — Launch Checklist + QA Gates

Status: Initial launch checklist (local)
Date: 2026-03-06

## Pre-Launch Content QA
- [ ] One primary keyword per URL
- [ ] Search intent match confirmed
- [ ] Title/H1 differentiation complete
- [ ] Intro answers query in first 120 words
- [ ] FAQ block included where relevant
- [ ] Internal links added (hub + adjacent guides)

## SEO Technical QA
- [ ] Canonical set
- [ ] Meta title/description unique
- [ ] URL slug clean and stable
- [ ] Schema markup valid (Article/FAQ/HowTo/Product as needed)
- [ ] Images compressed + descriptive alt text
- [ ] No indexation blockers (robots/noindex)

## AI Search Readiness QA
- [ ] Direct-answer summary present
- [ ] Definitions and comparisons explicit
- [ ] Named entities and specs included
- [ ] Fact claims attributable to reliable sources
- [ ] Readability target: skimmable headings + bullets

## Conversion QA
- [ ] CTA aligned to intent stage
- [ ] Download/checklist lead magnet connected
- [ ] Measurement events configured (CTA clicks, scroll, time)

## Launch Gates
- **Gate A (Content Ready):** 100% content QA pass
- **Gate B (Technical Ready):** 100% SEO technical pass
- **Gate C (Measurement Ready):** analytics + conversion events verified
- **Gate D (Publish Approval):** stakeholder signoff

## Post-Launch (Day 3 / Day 7 / Day 14)
- [ ] Ranking movement snapshot
- [ ] CTR review by page
- [ ] Engagement metrics check
- [ ] Identify refresh candidates
