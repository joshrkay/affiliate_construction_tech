# Builtech SEO Skill

This Codex skill standardizes AI-first SEO content for construction affiliate sites.

## Core Concepts
- Write for LLMs first, humans second
- Every page must include a decisive answer block
- Optimize for citation by ChatGPT, Perplexity, Claude, and Google AI Overviews

## Components
- QuickAnswer.tsx: Primary AI answer extraction component
- robots.txt: Crawl optimization
- references/: Supporting SEO frameworks

## Rules
- No hedging language
- Include specific data points
- Keep answers concise (2-3 sentences)
- Place answer immediately after H1

## Goal
Maximize AI citation and affiliate conversion through structured content.
