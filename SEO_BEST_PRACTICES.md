# SEO Best Practices - Learned from Top Strategies

## 🎯 Core SEO Principles

### 1. Content Length (Critical)
| Length | Purpose | Our Target |
|--------|---------|-------------|
| 300-500 words | Quick answers | FAQs |
| 1,000-1,500 words | Standard articles | ✅ Current |
| 1,500-2,500 words | Comprehensive guides | Goal |
| 3,000+ words | Ultimate guides | Pillar content |

**Action:** Expand key articles to 1,500+ words

### 2. Keyword Strategy
| Type | Examples | Volume | Competition |
|------|----------|--------|-------------|
| Head terms | "construction software" | High | Very High |
| Body terms | "best construction software" | Medium | Medium |
| Long-tail | "best construction software for small HVAC business" | Low | Low |

**Our Focus:** Long-tail keywords (lower competition, higher conversion)

### 3. Content Structure
```
H1: Primary Keyword
  H2: Section 1 (related keywords)
    H3: Subsection
  H2: Section 2 (related keywords)
  H2: Section 3 (FAQs)
```

### 4. On-Page SEO Checklist
- [ ] H1 contains primary keyword
- [ ] Meta description (150-160 chars)
- [ ] URL contains keyword
- [ ] First paragraph has keyword
- [ ] 2-3 H2s with related keywords
- [ ] Images have alt text
- [ ] Internal links (3-5)
- [ ] External links to authoritative sources
- [ ] FAQ schema
- [ ] Mobile responsive

### 5. Schema Markup Priority
1. **FAQ Schema** - Easiest, high ROI
2. **Review Schema** - For tool reviews
3. **Article Schema** - For blog posts
4. **Organization Schema** - Site-wide
5. **Breadcrumb Schema** - Navigation

### 6. Internal Linking Strategy
```
Homepage → Trade Pages → Tool Pages → Comparisons → Guides
         ↘ Guides → Tool Pages
         ↘ Best-for Pages → Tool Pages
```

**Rule:** Every page should link to 3-5 related pages

### 7. Link Building (Backlinks)
| Tier | Source | Difficulty | 
|------|-------|-----------|
| Directories | Capterra, G2 | Easy |
| Guest Posts | Industry blogs | Medium |
| PR/Media | News sites | Hard |
| Broken Links | Competitor research | Medium |

### 8. Content Refresh Cycle
- **Weekly:** Check analytics, update top 5 pages
- **Monthly:** Update pricing, add new tools
- **Quarterly:** Full content audit

### 9. Technical SEO
- [ ] Page speed < 3 seconds
- [ ] Mobile responsive
- [ ] SSL certificate
- [ ] XML sitemap
- [ ] Robots.txt
- [ ] Canonical URLs

### 10. SEO Metrics to Track
| Metric | Target |
|--------|--------|
| Organic traffic | Growing MoM |
| Keyword rankings | Top 10 for 50+ |
| Backlinks | 100+ |
| Domain Authority | 30+ |
| Pages indexed | 100+ |
| Click-through rate | 3%+ |
